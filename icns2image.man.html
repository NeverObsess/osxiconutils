NAME

     icns2image — Convert Apple icns file to image

SYNOPSIS

     icns2image [−vh] [−t format] source destination

DESCRIPTION

     icns2image is a command line tool to create an image from a Mac OS X icns
     file

     The following flags are supported:

     −r, −‐representation
              Specify representation size to convert (e.g. 512, 128, ’all’).

     −t, −‐type
              Specify output image format (e.g. jpeg, png, bmp, tiff)

     −v, −‐version
              Print program version and exit

     −h, −‐help
              Print help and exit

EXAMPLES

     icns2image ‐t jpeg source.icns source.jpg

     icns2image icon.icns icon.png
